# Firecrawl Design Analysis

Based on the Figma frame reference, key visual patterns to replicate:

## Layout Structure
- Clean, centered layout with generous whitespace
- Max-width container for content (~1200px)
- Consistent vertical spacing between sections
- Grid-based layouts for feature sections

## Typography
- Large, bold headlines (likely 48-64px)
- Clean sans-serif font (Geist or similar)
- Monospace font for code/technical elements (Geist Mono)
- Clear hierarchy with size and weight

## Color Palette (to adapt to dark + purple)
- Light background (will become dark: ~#0a0a0a)
- Orange accents (will become purple: ~#8b5cf6)
- Subtle borders and dividers
- High contrast text

## Design Elements
- Thin border lines creating grid patterns
- Rounded corners on cards/containers
- Subtle shadows on elevated elements
- Code block styling with syntax highlighting
- Decorative ASCII/matrix-style background elements

## Sections Observed
1. Hero with large headline and CTA
2. Feature grid (3 columns)
3. Product showcase with visuals
4. Comparison/stats section
5. Content/blog section
6. Contact/CTA section
7. Footer with links

## Spacing System
- Consistent padding (likely 16/24/32/48/64px scale)
- Section spacing (80-120px between major sections)
- Tight line-height for headlines
- Generous line-height for body text
